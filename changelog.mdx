## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added advanced search with boolean operators and field filtering
- Introduced custom themes and branding options
- Implemented automated publishing workflows with Git integration
- Added support for webhooks and real-time notifications

## Improvements

- Enhanced API performance with optimized database queries
- Improved user interface responsiveness on mobile devices
- Updated documentation editor with better Markdown support
- Added bulk operations for managing multiple pages

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Complete API redesign with RESTful endpoints
- Introduced workspaces for better organization
- Added collaboration features with comments and reviews
- Implemented version control for documentation pages

## Breaking Changes

- API endpoints have been restructured (see migration guide)
- Authentication method changed from API keys to Bearer tokens
- Page structure now requires workspace association

</Update>

- Corrected API pagination for large result sets
