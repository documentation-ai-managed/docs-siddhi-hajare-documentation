## API Fundamentals

Siddhi Hajare Documentation provides a powerful REST API that allows you to programmatically manage your documentation. Whether you're building custom integrations, automating workflows, or syncing with external systems, our API offers comprehensive endpoints for all platform features.

<Expandable title="Authentication" default-open="true">
  All API requests require authentication using Bearer tokens. Obtain your API key from the workspace settings and include it in the Authorization header.
</Expandable>

<Expandable title="Rate Limits" default-open="false">
  The API enforces rate limits to ensure fair usage. Free accounts have 100 requests per hour, while paid plans offer higher limits.
</Expandable>

<Expandable title="Error Handling" default-open="false">
  API responses include standard HTTP status codes. Check the response body for detailed error messages and resolution steps.
</Expandable>

  ```javascript
// Create a new page
const createPage = async (title, content) => {
  const response = await fetch('/api/v1/pages', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${process.env.API_TOKEN}`
    },
    body: JSON.stringify({
      title,
      content,
      workspaceId: 'workspace-123',
      tags: ['api', 'guide']
    })
  });
  
  if (!response.ok) {
    throw new Error(`API error: ${response.status}`);
  }
  
  return await response.json();
};

// Usage
const newPage = await createPage(
  'API Integration Guide',
  '# Introduction\n\nThis guide covers API integration.'
);
console.log('Page created:', newPage.id);
```
  ```python
import requests
from typing import Dict, Any

def create_page(api_token: str, title: str, content: str) -> Dict[str, Any]:
    """Create a new documentation page."""
    url = 'https://api.siddhidocs.com/v1/pages'
    headers = {
        'Content-Type': 'application/json',
        'Authorization': f'Bearer {api_token}'
    }
    data = {
        'title': title,
        'content': content,
        'workspaceId': 'workspace-123',
        'tags': ['api', 'guide']
    }
    
    response = requests.post(url, json=data, headers=headers)
    response.raise_for_status()
    
    return response.json()

# Usage
page = create_page(
    api_token='your-token-here',
    title='API Integration Guide',
    content='# Introduction\n\nThis guide covers API integration.'
)
print(f"Page created: {page['id']}")
```
