## Custom Themes and Branding

Transform your documentation with custom themes that match your brand identity. Siddhi Hajare Documentation supports extensive customization options for colors, fonts, and layouts.

<Steps>
  <Step title="Access Theme Settings" icon="settings">
    Navigate to workspace settings and select the Themes tab.
  </Step>
  <Step title="Configure Colors" icon="palette">
    Choose primary and secondary colors that align with your brand guidelines.
  </Step>
  <Step title="Upload Assets" icon="image">
    Add custom logos, favicons, and brand assets.
  </Step>
  <Step title="Preview and Apply" icon="eye">
    Preview your theme changes and apply them to your workspace.
  </Step>
</Steps>

<CodeGroup tabs="CSS,JSON">
  ```css
/* Custom theme CSS */
:root {
  --brand-primary: #3B82F6;
  --brand-secondary: #1E40AF;
  --text-primary: #1F2937;
  --text-secondary: #6B7280;
  --background-primary: #FFFFFF;
  --background-secondary: #F9FAFB;
}

.documentation-header {
  background-color: var(--brand-primary);
  color: white;
}

.page-content {
  color: var(--text-primary);
  background: var(--background-primary);
}
```
  ```json
{
  "theme": {
    "colors": {
      "primary": "#3B82F6",
      "secondary": "#1E40AF",
      "text": "#1F2937",
      "background": "#FFFFFF"
    },
    "fonts": {
      "heading": "Inter",
      "body": "Inter"
    },
    "layout": {
      "sidebarWidth": "280px",
      "maxWidth": "1200px"
    }
  }
}
```
</CodeGroup>
